---
title: "Qualification"
---

<p className="lede">Qualify hard, early. If there is no action-taking agent and no one who answers to an auditor, it is not a deal yet, it is a design-partner conversation.</p>

<table><tbody><tr><td>Metrics</td><td>Agents in prod, % actions auto-approved safely, time to answer an audit request, incidents avoided.</td></tr><tr><td>Economic buyer</td><td>CISO/VP Security on a call by week 2.</td></tr><tr><td>Decision criteria</td><td>In-path enforcement, latency SLO, policy coverage, <b>signed</b> audit evidence, EU hosting.</td></tr><tr><td>Decision process</td><td>30-45 day observe→enforce pilot, exec readout, security review.</td></tr><tr><td>Paper / procurement</td><td>Security review, DPIA/DORA mapping, data-residency sign-off.</td></tr><tr><td>Implication of pain</td><td>A named incident or an audit finding on agent actions; board question on file.</td></tr><tr><td>Champion</td><td>Owns the agent runtime (MCP/gateway/platform); can wire one path in a day.</td></tr><tr><td>Competition</td><td>Build-it-yourself, a gateway/identity vendor, or "we'll wait". See Landscape.</td></tr></tbody></table>
